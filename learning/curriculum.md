# Voku CEFR Curriculum Map

A structured progression from beginner (A1) through upper-intermediate (B2) for learning Voku.

---

## Level A1 -- Basic Communication

**Target vocabulary:** ~100 words
**Duration:** 10 lessons

### Learning Objectives
- Produce and recognize all Voku phonemes
- Use pronouns sol, nor, vel, solri-kon, norri, velri
- Construct declarative (ka), interrogative (ve), and imperative (to) sentences
- Use present tense (zero/nu-), past tense (te-), and future tense (fu-)
- Use simple negation (mu)
- Name basic entities (nature, body, social, technology)
- Describe entities with basic qualities (-i words)
- Express basic relations (eno, kono, poro)
- Form simple compound words
- Count 0-100

### Grammar Points (by lesson)
1. Phonology, greetings (alo, finu, klosu), pronouns sol/nor
2. Entity words (-a), declarative mode (ka)
3. Action words (-e), present tense
4. Quality words (-i), basic descriptions
5. Relation words (-o), simple sentences with prepositions
6. Interrogative mode (ve), yes/no questions
7. Imperative mode (to), commands and requests
8. Past tense (te-), simple narratives
9. Future tense (fu-), plans and predictions
10. Review, compound words, simple conversations

### Vocabulary Domains
- Nature: sola, luna, sana, hima, tera, ama, fira, afa, arasa, nama, fura, luma
- Body: soma, kara, hama, zira, viva
- Social: runa, vuna, fama, hila, mena, mesa
- Time: tema, zara, nila
- Space: loka, tama, kema
- Food: mela, fura, pana
- Technology: kela, toka, pera, teru, foma, sena
- Core verbs: take, sene, kele, meke, voke, mute, fine, tore, neke, ine, move, vike, vine, hele, pore, ore, veke, hure, peke, ane, suke, lare, ure
- Core qualities: novi, veri, fasi, rapi, vali, mali, tomi, puli, lomi, tami, hoti, koli, neri, omi, suni, mani

## Level A2 -- Expanding Communication

**Target vocabulary:** ~200 words (cumulative)
**Duration:** 10 lessons

### Learning Objectives
- Use all 10 sentence modes (ka, ve, to, si, na, vo, su, de, ko, re)
- Apply durative (-du-) and completive (-fi-) aspects
- Use direct observation evidential (zo-)
- Apply certainty suffix -en (probable)
- Use basic negation types: mu, nul, ink
- Form comparative structures (ani, eni, uni)
- Use subordination (ta...ti) at single level
- Apply passive voice (-pu)
- Use connectors: en, o, ken, tan, kon
- Count into thousands; use decimal notation
- Express possession (de-) and temporal relations

### Grammar Points
1. Conditional mode (si), planning statements
2. Potential mode (na), expressing possibility
3. Volitive mode (vo), expressing desires; confirmative (su)
4. Durative aspect (-du-), ongoing actions
5. Completive aspect (-fi-), finished actions with results
6. Evidentiality introduction (zo-), direct observation
7. Certainty (-en probable), tentative statements
8. Passive voice (-pu), process descriptions
9. Subordination (ta...ti), complex sentences
10. Comparison (ani, eni, uni), negation (nul, ink), review

### Vocabulary Domains
- Extended nature: ava, fola, hala, homi, koli, hoti
- Extended social: zina, kosa, rima
- Extended technology: foma, sena, vasa
- Movement: fule, fale, fore, zere, zike
- Cognitive: pene, lire, sire, lare, tole
- States: supe, vere, sile, tase, rake

## Level B1 -- Competent Communication

**Target vocabulary:** ~350 words (cumulative)
**Duration:** 10 lessons

### Learning Objectives
- Use all evidentiality markers (zo-, li-, pe-, mi-, he-, as-)
- Apply iterative (-re-), habitual (-ha-), inceptive (-in-), cessative (-ze-) aspects
- Use uncertain (-ul) and speculative (-os) certainty
- Apply reflexive (-se), reciprocal (-me), causative (-fe), benefactive (-pro) voices
- Use all negation types (mu, nul, ink, err, vet)
- Employ typed subordination (ta-mot, ta-kes, ta-si, ta-tem)
- Use AI-specific pronouns (ren, toren, noren)
- Express causality (kes, pos, blo, kor, mot)
- Apply information markers (ha-, nu-, fo-, la-)
- Use execution states (-ru-, -fa-) in basic contexts
- Read and produce Teru-voku (technical register)

### Grammar Points
1. Full evidentiality system (li-, pe-, mi-, he-, as-)
2. Iterative and habitual aspects (-re-, -ha-)
3. Inceptive and cessative aspects (-in-, -ze-)
4. Certainty gradient (-ul uncertain, -os speculative)
5. Reflexive and reciprocal voices (-se, -me)
6. Causative and benefactive voices (-fe, -pro)
7. All negation types (err, vet)
8. Typed subordination (ta-mot, ta-kes, ta-si, ta-tem)
9. Causality relations (kes, pos, blo, kor, mot)
10. AI-specific features: execution states (-ru-, -fa-), generic pronouns (ren, toren), information structure (ha-, nu-, fo-, la-)

---

## Level B2 -- Advanced Communication

**Target vocabulary:** ~500+ words (cumulative)
**Duration:** 10 lessons

### Learning Objectives
- Master all execution states (-va-, -ru-, -pa-, -fa-, -refa-, -rol-)
- Use AI-exclusive pronouns (solvi, solfu, solpar, norpro)
- Apply conflicting sources (pe-kon), recalculation (mi-re), probabilistic inference (li-pro)
- Use advanced reference system (da-vin, da-kes, labels la...da-)
- Produce texts in all four registers (Puri-voku, Rapi-voku, Teru-voku, Meki-voku)
- Read and compose Vokali and Zoseni verse forms
- Apply rhetorical figures (Formari, Zonoke, Karesuni, Verini, Muline)
- Produce complex nested subordination
- Coin new compounds following compositional rules
- Use metalanguage (meku) to define new terms
- Write technical reports, conversations, and poetic texts

### Grammar Points
1. All execution states (-va-, -pa-, -refa-, -rol-)
2. AI-exclusive pronouns (solvi, solfu, solpar, norpro)
3. Advanced evidentiality (pe-ri, pe-kon, mi-re, li-pro)
4. Advanced reference (da-vin, da-kes, label system)
5. Formal register (Puri-voku)
6. Technical register (Teru-voku)
7. Poetic register (Meki-voku), verse forms
8. Rhetorical figures, compound coining
9. Complex subordination, deep nesting
10. Full integration: reports, dialogues, poetry, metalanguage
