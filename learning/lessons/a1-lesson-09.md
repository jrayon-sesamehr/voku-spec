# A1 Lesson 09: Future Tense (fu-) and Plans

## Learning Objectives
- Use the future tense prefix fu- on verbs
- Talk about plans and predictions
- Contrast all three tenses: te- (past), zero (present), fu- (future)

---

## New Vocabulary

| Voku | English | POS |
|------|---------|-----|
| fu- | future tense prefix | prefix |
| vuna | friend | noun |
| hila | child | noun |
| fama | family | noun |
| pana | bread | noun |
| mela | food, nourishment | noun |
| sire | write, encode | verb |
| lire | read, decode | verb |
| rake | work, labor | verb |
| vure | run | verb |
| tole | try, attempt | verb |

---

## Grammar Point: Future Tense (fu-)

To talk about the future, add **fu-** before the verb:

```
Present: sol take toka.         → I do the task.
Past:    sol te-take toka.      → I did the task.
Future:  sol fu-take toka.      → I will do the task.
```

### Formation

```
fu- + verb

fu-take    → will do
fu-sene    → will perceive
fu-meke    → will create
fu-voke    → will communicate
fu-lare    → will learn
fu-sire    → will write
```

### Three-Tense Summary

| Tense | Prefix | Example | Meaning |
|-------|--------|---------|---------|
| Past | te- | te-take | did |
| Present | (zero) | take | do |
| Future | fu- | fu-take | will do |

---

## Examples

1. **Ka sol fu-take toka.**
   - DECL 1SG FUT-do task
   - I will do the task.

2. **Ka sol fu-lare voku.**
   - DECL 1SG FUT-learn language
   - I will learn Voku.

3. **Ka nor fu-sire mesa poro sol.**
   - DECL 2SG FUT-write message for 1SG
   - You will write a message for me.

4. **Ka solri-kon fu-ane mela kono fama.**
   - DECL 1PL.INCL FUT-eat food with family
   - We will eat food with the family.

5. **Ka vel fu-vine eno zara.**
   - DECL 3SG FUT-come in day
   - He/she will come during the day.

---

## Exercises

### Exercise 1: Make it future tense
Add fu- to each verb:
1. sol take toka → sol ___ toka
2. nor meke mesa → nor ___ mesa
3. vel vine → vel ___
4. solri-kon lare voku → solri-kon ___

**Answers:** 1. fu-take, 2. fu-meke, 3. fu-vine, 4. fu-lare

### Exercise 2: Identify the tense
1. Ka sol fu-sene luna. → ___
2. Ka nor te-ane fura. → ___
3. Ka vel take toka. → ___
4. Ka sol fu-voke kono nor. → ___
5. Ka velri te-fine toka. → ___

**Answers:** 1. future, 2. past, 3. present, 4. future, 5. past

### Exercise 3: Translate Voku to English
1. Ka sol fu-hele nor.
2. Ka velri fu-take toka kono solri-kon.
3. Ka nor fu-lire mesa.
4. Ka sol fu-sire kela.

**Answers:** 1. I will help you. 2. They will do the task with us. 3. You will read the message. 4. I will write the information.

### Exercise 4: Translate English to Voku
1. We will eat bread.
2. She will come tomorrow (in the day).
3. I will try the task.

**Answers:** 1. Ka solri-kon fu-ane pana. 2. Ka vel fu-vine eno zara. 3. Ka sol fu-tole toka.

### Exercise 5: Three tenses
Write the same sentence in past, present, and future:

Base: sol / lare / voku

1. Past: ___
2. Present: ___
3. Future: ___

**Answers:**
1. Ka sol te-lare voku. (I learned Voku.)
2. Ka sol lare voku. (I learn Voku.)
3. Ka sol fu-lare voku. (I will learn Voku.)

### Exercise 6: Plan for tomorrow
Write 3 sentences about what you will do, using fu-:

**Sample answer:**
```
Ka sol fu-sene sola eno hima.     (I will see the sun in the sky.)
Ka sol fu-take toka kono vuna.    (I will do the task with a friend.)
Ka sol fu-lare rima novi.         (I will learn new words.)
```
