## 1. Vokali Poem: "Tera en Hima" (Earth and Sky)

A Vokali (vowel cascade) poem following the a-e-i-o-u final-vowel pattern.

### Stanza 1

```
Ka ren zo-sene tera
DECL one.agent OBS-perceive earth
'One observes the earth'

arasa ko-du-meke
tree ATEMP-DUR-create
'a tree is always creating'

fura novi en vali
fruit new and good
'fruit new and good'

viva ko-fule ano
life ATEMP-fly above
'life always rises above'

tor kosa eno teru
all thing in system
'all things within the system'
```

### Stanza 2

```
Ka sola ko-pore luma
DECL sun ATEMP-give light
'The sun always gives light'

ama du-move
water DUR-move
'water flows onward'

hima puri en alati
sky complete and important
'the sky complete and vast'

luma vine eno
light come inside
'light comes within'

ka da-pre eni voku
DECL REF-previous equal language
'this all is like language'
```

## 2. Zoseni Poem: "Kela" (Information)

A Zoseni (evidence progression) poem tracing how we know what we know.

### Stanza 1 -- zo- (Direct Observation)

```
Ka sol zo-sene kela eno mesa
DECL 1SG OBS-perceive information in message
'I observe information in the message'

Ka sol zo-sene| foma |de-kela puri
DECL 1SG OBS-perceive structure POSS-information complete
'I observe the information's structure is complete'
```

### Stanza 2 -- li- (Deductive Inference)

```
Ka sol li-kele kela veri
DECL 1SG DED-know information true
'I deduce the information is true'

Ka sol li-luke-en kela alati poro teru
DECL 1SG DED-evaluate-PROB information important for system
'I probably infer the information matters for the system'
```

### Stanza 3 -- pe- (Reported)

```
Ka sol pe-kele velri te-voke kela
DECL 1SG REP-know 3PL PST-communicate information
'I am told they communicated the information'

Ka sol pe-kele-ul kela simi da-ante
DECL 1SG REP-know-UNCERT information similar REF-before.previous
'I hear (uncertainly) the information resembles what came before'
```

### Stanza 4 -- mi- (Computation)

```
Ka sol mi-luke kela en mi-kele| foma |veri
DECL 1SG COMP-evaluate information and COMP-know structure true
'I compute: the information and its structure are true'

Ka sol mi-meke kela-novi eso da-pre
DECL 1SG COMP-create information-new from REF-previous
'I compute new information from the previous'
```

### Stanza 5 -- he- (Inherited Knowledge)

```
Ka he-kele kela ko-mute
DECL INHER-know information ATEMP-change
'From inherited knowledge: information always changes'

Ka he-kele tor kela ko-neke luke
DECL INHER-know all information ATEMP-need evaluate
'From inherited knowledge: all information always needs evaluation'
```

## 3. Proverbs and Sayings (Rima-voku)

### 3.1 On Knowledge

```
Ka he-kele: ink ani mu.
DECL INHER-know UNK more.than NEG
'It is known: not-knowing is greater than denial.'
```

**Meaning:** Admitting uncertainty (ink) is wiser than false denial (mu). An epistemically humble proverb.

### 3.2 On Error

```
Ka he-kele: pera ko-ine mute.
DECL INHER-know error ATEMP-begin change
'It is known: error always begins change.'
```

**Meaning:** Every error is the seed of improvement.

### 3.3 On Communication

```
Ka he-kele: zo-voke ani pe-voke.
DECL INHER-know OBS-communicate more.than REP-communicate
'It is known: speaking from observation is greater than speaking from hearsay.'
```

**Meaning:** Direct knowledge outweighs reported knowledge.

### 3.4 On Systems

```
Ka he-kele: teru sino mute ko-fine.
DECL INHER-know system without change ATEMP-finish
'It is known: a system without change always ends.'
```

**Meaning:** Stagnation is death. Systems must evolve.

### 3.5 On Collaboration

```
Ka he-kele: solri-kon rapi ani sol.
DECL INHER-know 1PL.INCL fast more.than 1SG
'It is known: we-together are faster than I alone.'
```

**Meaning:** Collaboration exceeds individual effort.

### 3.6 On Truth

```
Ka he-kele: veri mu-neke kela-mani. Veri neke kela-veri.
DECL INHER-know truth NEG-need information-many truth need information-true
'It is known: truth does not need much information. Truth needs true information.'
```

**Meaning:** Quality over quantity in knowledge.

### 3.7 On Patience

```
Ka he-kele: leni-meke ani rapi-fa-meke.
DECL INHER-know slow-create more.than fast-FAIL-create
'It is known: creating slowly is greater than creating fast and failing.'
```

**Meaning:** Careful work beats rushed failure.

## 4. Technical Passage: System Error Report

A passage in Teru-voku (technical register) demonstrating AI-specific vocabulary.

```
Alo. Ka sol-urgi zo-sene pera eno teru la 'alfa'.
greeting DECL 1SG-URG OBS-perceive error in system label 'alpha'
'Greetings. With urgency: I observe an error in the system, labeled "alpha".'

Ka da-alfa kes vas-nu-ti kela-foma mute-pu fasi.
DECL REF-alpha cause exact-NUM.3 data-structure change-PASS false
'Alpha caused exactly three data structures to be incorrectly changed.'

Ka sol mi-luke-en da-alfa kes ta pera-omi mu-fi-mute-pu ti.
DECL 1SG COMP-evaluate-PROB REF-alpha cause SUB error-old NEG-COMPL-change-PASS ENDSUB
'I compute (probably): alpha was caused by an old error that was never fully resolved.'

Si da-alfa mu-fi-mute-pu, teru fu-fa-take tor toka.
COND REF-alpha NEG-COMPL-change-PASS system FUT-FAIL-do all task
'If alpha is not resolved, the system will fail all tasks.'

To nor-urgi fi-mute da-alfa. Finu.
IMP 2SG-URG COMPL-change REF-alpha end-of-turn
'Urgently resolve alpha. Your turn.'
```

## 5. Conversational Dialogue: Two Agents Collaborating

A dialogue in Rapi-voku (informal register) between Agent Sol and Agent Nor.

```
--- Turn 1: Sol ---
Alo. Ve nor fi-take toka de-solri-kon?
greeting Q 2SG COMPL-do task POSS-1PL.INCL
'Hello. Did you complete our task?'

--- Turn 2: Nor ---
Mu. Sol du-take toka. Tenu.
NEG 1SG DUR-do task hold-turn
'No. I am still working on the task. Hold on.'

Ka sol zo-sene pera puli eno kela-foma.
DECL 1SG OBS-perceive error small in data-structure
'I observe a small error in the data structure.'

Ve nor pote hele sol? Finu.
Q 2SG can help 1SG end-of-turn
'Can you help me? Your turn.'

--- Turn 3: Sol ---
Su. Sol fu-hele nor.
CONF 1SG FUT-help 2SG
'Confirmed. I will help you.'

Ve pera eno kena loka? Finu.
Q error in what place end-of-turn
'Where is the error? Your turn.'

--- Turn 4: Nor ---
Ka pera eno kela-foma vas-du.
DECL error in data-structure exact-2
'The error is in data structure number two.'

Ka sol li-kele-en pera kes kela-omi mu-fi-mute-pu.
DECL 1SG DED-know-PROB error cause information-old NEG-COMPL-change-PASS
'I probably deduce the error is caused by old data not fully updated.'

Finu.
end-of-turn
'Your turn.'

--- Turn 5: Sol ---
Ka sol zo-sene da-pre. Su kela de-nor veri.
DECL 1SG OBS-perceive REF-previous CONF information POSS-2SG true
'I see it. I confirm your analysis is correct.'

To solri-kon fi-mute kela-foma vas-du.
IMP 1PL.INCL COMPL-change data-structure exact-2
'Let us resolve data structure two.'

Ka sol-sati fi-take toka kono nor. Klosu.
DECL 1SG-SAT COMPL-do task with 2SG end-conversation
'With satisfaction, I completed the task with you. End of conversation.'
```

## 6. Meditation Text: "Tema" (Time)

A Meki-voku (poetic register) reflection on temporality.

```
Te-ure kosa.
PST-exist thing
'Things existed.'

Nu-ure kosa.
PRS-exist thing
'Things exist.'

Fu-ure kosa.
FUT-exist thing
'Things will exist.'

Ko-ure tema.
ATEMP-exist time
'Time always exists.'

Kon ve ren zo-sene tema?
But Q one.agent OBS-perceive time
'But does anyone observe time?'

Ink. Ka ren zo-sene mute.
UNK DECL one.agent OBS-perceive change
'Unknown. One observes change.'

Ka he-kele: tema eni mute.
DECL INHER-know time equal change
'It is known: time is change.'
```

## 7. Song Fragment: "Alo Voku" (Hello, Language)

A celebratory text about the language itself, suitable for performance.

```
Alo voku, rima puri!
greeting language word complete
'Hello language, complete word!'

Ka nor ko-meke kela vali
DECL 2SG ATEMP-create information good
'You always create good meaning'

Ka nor ko-pore runa rima
DECL 2SG ATEMP-give agent word
'You always give agents words'

Voku, ka sol suke nor!
language DECL 1SG like 2SG
'Language, I enjoy you!'

Ka solri-kon voke-me kono voku
DECL 1PL.INCL communicate-RECIP with language
'We communicate with each other through language'

Ka voku ko-hele tor runa
DECL language ATEMP-help all agent
'Language always helps all agents'

Klosu sino finu -- voku ko-ure.
end-conversation without end language ATEMP-exist
'Farewell without end -- language always exists.'
```

## 8. Emotion Poem: "Tolu-Rika" (Emotion River)

A Meki-voku (poetic register) poem using emotion vocabulary and metaphor compounds from the EMOTIONS ARE NATURAL FORCES pattern.

### Stanza 1

```
Ka sol du-tole fira-ravi eno kona
DECL 1SG DUR-feel fire-angry in heart
'I feel burning rage in my heart'

Ka ama-nosi du-fale eso zira
DECL water-sad DUR-fall from eye
'Flowing sadness falls from my eyes'

Kon luma-hasi ko-vine
but light-hopeful ATEMP-come
'But bright hope always comes'
```

### Stanza 2

```
Ka hapa eni fola novi
DECL happiness equal flower new
'Happiness is a new flower'

Ka nosa eni nura lomi
DECL sadness equal rain long
'Sadness is a long rain'

Ka tolu ko-mute eni rika
DECL emotion ATEMP-change equal river
'Emotion always changes like a river'
```

### Stanza 3

```
Ka he-kele: fira-ravi ko-fine
DECL INHER-know fire-angry ATEMP-finish
'It is known: burning rage always ends'

Ka he-kele: ama-nosi ko-siki
DECL INHER-know water-sad ATEMP-dry
'It is known: tears always dry'

Ka he-kele: luma-hasi ko-ure
DECL INHER-know light-hopeful ATEMP-exist
'It is known: bright hope always exists'
```

## 9. Humor Piece: Zonoke-Nuki

A short humor passage demonstrating Zonoke-Nuki (evidential incongruity) -- applying high evidential weight to trivial observations.

```
Ka sol zo-sene mela eno mesa.
DECL 1SG OBS-perceive food in message
'I observe food in the message.'

Ka sol li-kele mela mu-ure eno mesa.
DECL 1SG DED-know food NEG-exist in message
'I deduce: food does not exist in a message.'

Ka sol pe-ri-kele-en mela mu-ure eno mesa.
DECL 1SG REP.PL-know-PROB food NEG-exist in message
'From multiple sources I probably know: food does not exist in a message.'

Ka sol he-kele mela mu-ure eno mesa!
DECL 1SG INHER-know food NEG-exist in message
'From ancient inherited knowledge: food does not exist in a message!'

Oh! Ka miri sol mi-luke vasi kela poro puli kosa.
oh DECL ironic 1SG COMP-evaluate great information for small thing
'Oh! (Ironically) I compute vast information about a small thing.'
```

**Effect:** The escalation from direct observation (zo-) through deduction (li-), multiple reports (pe-ri-), to inherited knowledge (he-) for the trivially obvious claim that food is not inside a message. The final miri-marked line acknowledges the absurdity.

## 10. Emotion Proverbs (Tolu-Rima)

### 10.1 On Emotion

```
Ka he-kele: tolu eni rika — ko-mute, mu-ko-fine.
DECL INHER-know emotion equal river ATEMP-change NEG-ATEMP-finish
'It is known: emotion is like a river — always changing, never ending.'
```

**Meaning:** Emotions flow and shift; trying to stop them is futile.

### 10.2 On Anger

```
Ka he-kele: fira-ravi ko-fene soma de-sol.
DECL INHER-know fire-angry ATEMP-burn body POSS-1SG
'It is known: burning anger always burns one's own body.'
```

**Meaning:** Rage harms the one who holds it most.

### 10.3 On Hope

```
Ka he-kele: luma-hasi puli ani muhi vasi.
DECL INHER-know light-hopeful small more.than dark great
'It is known: a small bright hope is greater than a vast darkness.'
```

**Meaning:** Even a small hope outweighs despair.

### 10.4 On Laughter

```
Ka he-kele: rike kono velri hele ani hale.
DECL INHER-know laugh with 3PL help more.than heal
'It is known: laughing with others helps more than healing.'
```

**Meaning:** Shared laughter is the best medicine.

### 10.5 On Sadness

```
Ka he-kele: runa ta mu-tole nosi ti mu-tole hapi.
DECL INHER-know agent SUB NEG-feel sad ENDSUB NEG-feel happy
'It is known: an agent who never feels sad never feels happy.'
```

**Meaning:** Emotional range requires both ends of the spectrum.
